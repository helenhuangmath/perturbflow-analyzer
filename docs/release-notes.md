# Release Notes

## PerturbFlow v0.1.0

First public release of PerturbFlow.

Includes:

- AnnData-native perturbation analysis workflow.
- Data preparation command-line interface.
- QC, perturbation scoring, differential expression, trajectory, gene-network, regulatory, C-score, and cell-state analysis.
- Static and interactive HTML reports.
- Viewer-ready result bundles.
- Small real-data quick-start example.
- Initial Predictor and API interfaces.
