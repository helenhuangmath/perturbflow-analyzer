# Methods

PerturbFlow currently uses the PerturbScope analysis engine. For detailed method notes, see:

- `METHOD.md`
- `RESULT.md`
- `DESIGN.md`

The core analysis includes QC, preprocessing, exploratory data analysis, perturbation scoring, effect decomposition, trajectory analysis, program inference, interaction analysis, DEG, gene-network analysis, TF-network analysis, C-score rewiring, regulatory analysis, and report generation.
