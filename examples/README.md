# PerturbFlow Example Notebooks

These notebooks show common user workflows:

- `01_prepare_and_run.ipynb`: prepare an AnnData file and run the full pipeline.
- `02_step_rerun_and_config.ipynb`: customize configuration and rerun selected steps.
- `03_interpret_with_agents.ipynb`: create AI/agent-ready interpretation files.
- `04_explore_outputs.ipynb`: inspect result tables, plots, reports, and bundle files.

The notebooks use placeholder input paths by default. Update the paths at the top of each notebook before running an analysis.
